## 开始使用
1. 添加CSS 样式
    使用本框架前请在CSS中开启 box-sizing
    ```
    *,
    *::before,
    *::after{
        margin: 0;
        padding: 0;
        box-sizing: border-box;
    }
    ```
    默认样式：
    ```
    :root {
    --button-height: 32px;
    --font-size: 14px;
    --button-bg: white;
    --button-active-bg: #eee;
    --border-radius: 4px;
    --color: #333;
    --border-color: #999;
    --border-color-hover: #666; 
    }
    ```
    IE 15以上
2. 安装 vue-demo
3. 引入 vue-demo
4. 引入svg-symbols
